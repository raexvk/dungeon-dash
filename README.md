# Dungeon Dash

Multiplayer dungeon-crawler browser game with real-time co-op, built with pure Canvas rendering — no game frameworks.

## [Play Live](https://dungeon-dash.raexvk.workers.dev/)

## Tech Stack

- **Frontend** — React 19, TypeScript, Vite, Canvas API
- **Multiplayer** — WebSockets via Cloudflare Durable Objects
- **Hosting** — Cloudflare Workers

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Deploys to Cloudflare Workers via Wrangler.
